Cider

Getting Started

Cider is a fully managed development environment for open source hacking on Mac OSX. It's powered by rvm, homebrew and chef. You only need Xcode to get started.

Cider builds everything up in ~/Developer. It won't stomp on any of your current installations so you don't have to commit your entire machine to cider immediately. It's a breeze to rollback if you really want to.

You're gonna have to trust me here, bootstrap yourself from curl:

% curl \
   http://github.com/atmos/cider/raw/master/bootstrap.sh \
   -o - | sh

When this completes you'll probably want to launch a new shell. Hit ⌘t and you should be running under a cider environment.

Running Cider Normally

After the initial bootstrapping you only need to run the cider executable. You should run cider regularly to keep up with the latest changes. Cider will automatically update itself if the recipes tell it to. Currently there's no command line options, it just installs and exits. 

% cider
INFO: Starting Chef Solo Run
INFO: Replacing the run_list with ["homebrew", "rvm", "node"] from JSON

It will take a while on the first run but it's quick after that. If everything executes properly you'll see messages coming from chef during the cider run. Right now the recipes are hard coded but I hope to add support to allow developers to build their own flavors for their machines. Ideally everyone just picks what they want on their machines and excludes the stuff they don't want.

This might not be what you need right now, but if you have to bootstrap a new environment I suggest giving it a try. The entire environment should be able to be thrown away and recreated with no more than a single shell call to the cider executable.

What It Gives You

  • mysql running as your user
  • postgres daemon running as your user
  • mongodb running as your user
  • rvm with rubinius and latest stable ruby.
  • node.js development environment

Notes

  • Backups should be made for any dotfiles cider touches.
  • Customizing PS1/Shell Env/etc: 
    cinderella% cat ~/.cider.profile.custom
set -o vi
if [[ -x ~/Developer/bin/fortune ]]; then
  echo ""; fortune; echo ""
fi
alias nerd="vim +NERDTree"
alias g="git"
  • Sepcifying the default RVM Environment:
    cinderella% DEFAULT_RVM_VM=ruby-1.9.2 cider
INFO: Starting Chef Solo Run
